# QA Portfolio
Портфолио QA Engineer – Кирилл Денисов

Объект тестиравания: веб-сайт https://lesta.ru/.

## Содержание портфолио (создано в FireTMS):
1. Тест-кейсы (3 шт.):
   - LESTA-3: Успешная авторизация пользователя.
   - LESTA-4: Негативная авторизация (пустое поле E-mail).
   - LESTA-5: Проверка выпадающего списка "Игры" и отображения информации об играх.

2. Баг-репорт (1 шт.):
   - LESTA-6: Пустой запрос с пробелом в поле "Поиск по вакансиям" (переход на страницу результатов, хотя поле пустое).

3. Чек-листы (2 шт.):
   - Чек-лист проверки хэдера.
   - Чек-лист проверки страницы "ИГРЫ".

## Инструменты, использованные при создании:
- FireTMS - для оформления тест-кейсов, баг-репорта, чек-листов.
- Браузер Chrome - для выполнения проверок и создания скриншотов.
